# eslint-config-re-taro
Shareable ESLint config for Rintaro.

## Usage

### For JavaScript
```yaml
extends:
  - re-taro
```

### For TypeScript
```yaml
extends:
  - re-taro
  - re-taro/typescript
```

### For React
```yaml
extends:
  - re-taro
  - re-taro/react
```

### For Preact
```yaml
extends:
  - re-taro
  - re-taro/preact
```

### For Jest
```yaml
extends:
  - re-taro
  - re-taro/jest
```

### For Prettier
```yaml
extends:
  - re-taro
  - re-taro/prettier
```

### For Next.js
```yaml
extends:
  - re-taro
  - re-taro/react
  - re-taro/next
```

### For NestJS
```yaml
extends:
  - re-taro
  - re-taro/nest
```
